Known Issues/FAQ:

Task Killers

Important: Wifi Fixer's wifi health monitor service can be rendered inoperable by task managers and auto task killers.

It is strongly recommended that you do not use Wifi Fixer with this kind of software, but if you must, add WFMonitorService to the allowed list.

Too Many Networks

Important: Having too many Known Network entries can slow down your phone and burn more battery.

This is an Android issue.   If you have more than 10, try to remove unused or infrequently used networks.


Unable To Scan for Networks or Wifi Not Turning On

On the original Samsung Galaxy and several other devices, Wifi may become unable to turn on, with or without the message " Unable To Scan For Networks " displaying.

The issue is in the vendor implementation of the Android Framework interacting with the Wifi driver

It can be worked around either by rebooting the device or going into Settings:Privacy and doing a factory reset. If a full wipe factory reset doesn't work the problem may be hardware.

Note: You will lose all your settings if you perform a factory reset.

I wish that I could fix this problem, but it requires a vendor fix.

Rebooting or factory resetting is a temporary fix. In order to solve the problem permanently, you must update your device's software or root and install a custom rom.


Wifi Sleep Policy

On some devices, Wifi Sleep Policy appears to be ignored when the screen is off.

This is caused by wifi power save mode, which is not compatible with some wifi access points.

There is a workaround, "Nexus One Sleep Fix", in Advanced Options which may help.

Note: The workaround will temporarily disrupt the wifi connection when your screen goes off.

Set Wifi Usability to Max Wifi before enabling the workaround, and set Wifi Sleep Policy to Never

Cyanogen Mod 6 and above have a driver fix for this issue:
Cyanogen Mod Homepage

Known Networks

The Known Networks panel may not show any networks when wifi is disabled.

The network list cannot be loaded when wifi is off.

Other Wifi Apps

Wifi Fixer may conflict with other apps which use the Android wifi manager.   Try disabling the service when using conflicting apps if you have an issue.
There is no way to prevent other apps from accessing WifiManager.

Hidden SSIDs

Android is not able to work properly with wifi networks using hidden SSIDs.

If your network is set to non-broadcast SSID, you will have to change it to broadcast.

Why does the widget update so slowly?

The widget updates less frequently than the status notification to save system resources.

Why can't I change time between scans/checks?

These values can have a profound effect on battery life.  The timing has been balanced for maximum responsiveness with minimal battery use.

Why won't my Android 3.2+ device connect to a non-known open network using Wifi Fixer?
Google made changes to WifiConfiguration in Android 3.2 which causes networks created with a new WifiConfiguration object not to work properly due to missing fields.  I worked around the issue partially with Java reflection, but open networks fail on some builds of Android.  This will be fixed in an upcoming release of Wifi Fixer.
The bug report is here: Bug Report #21469
 

Why can't I add my network using Local Networks?
At this time Open/WPA/WPA2 is supported in Local Networks. You can use the Android settings app to connect.  The network will appear in Wifi Fixer under Known Networks.
There will be support for extended authentication networks in an upcoming release of Wifi Fixer.
