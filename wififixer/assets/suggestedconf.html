Configuring your Wifi Access Point

Channels and Protocols:
  • Protocols, by order of preference
    •
    1. AC
    2. N
    3. A
    4. G
    5. B
    If you can, set to just one protocol. Mixed modes can cause problems.

    Do not use 40mhz channel width on 2.4ghz N networks. Use 40mhz channel width on 5ghz N networks.


  • Encryption, by order of preference:
    1. WPA2-AES
    2. WPA
    3. WEP 128

  • Short Guard Interval (N only)
    •  If you are dropping connection too often, set this to off.

  • Long Preamble
    • Set this to enabled: it allows the phone's low power radio states to remain connected.

  • Channels
    • Channels 6 tends to be default.  If you are having interference problems (lots of drops, difficulty with initial connection) try setting to channel 1 or 11.

    Caveats:

  • Hidden SSID
    • Most Android devices will not operate correctly with hidden networks.  You must enable SSID broadcast for Android compatibility.

  • Ad Hoc
    • Non-infrastructure (peer to peer) wifi networks aren't supported at this time by Android.

  • Non-RFC SSID
    • Android works better when the SSID (the name of the Wifi Access Point) has no spaces and contains only Alphanumeric characters.
